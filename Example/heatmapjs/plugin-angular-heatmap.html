说明文档

安装

在heatmap.js下面添加angular-heatmap.js（在网页头部或在关闭body之前），并在应用中注入heatmap模块。一个heatmap指令可以采用以下属性： id、data、config、width、height，data是典型的heatmap.js数据对象，config是典型的配置对象。

API

angular-heatmap.js插件输出一个名为$heatmap的服务，可以直接访问你的热力图实例。
  • getInstance(id)
    返回id=[id]的热力图实例对象

  • getAllInstances()
    返回所有存在的热力图实例对象

示例

点击查看 在线示例. 

// 完整的指令用法范例
// <heatmap id="myHeatmapInstance" data="myHeatmapData" config="myHeatmapCfg"></heatmap>

// => heatmap 指令将自动对 myHeatmapData ＆ myHeatmapCfg 的更改做出反应。
// => 你可以通过 $heatmap.getInstance('myHeatmapInstance') 手动访问 heatmap 实例
