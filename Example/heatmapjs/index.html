heatmap.js

基于Web的动态热力图


heatmap.js 是一个轻量级、使用简单的 JavaScript 库，可帮助你可视化三维数据！

使用它为你的项目添加新的数据，建立一个基于它的业务、研究和可视化用户行为，或者，为什么不建立一个完全疯狂/炫酷的东西呢？

不管你想要对用户行为进行综合概述，还是查看分布在地图上的数千个事件...
heatmap.js 可以满足你所有的热力图可视化需求！

为什么使用 heatmap.js？

因为它是互联网上最先进的热力图可视化库。

它具有轻量级特性（压缩版大约3KB大小），并且刚刚发布了heatmap.js 2.0版本。 比任何时候都更加：
  • 快速，使用新的渲染模式。
  • 稳健, 可处理更多数据点（4万点以上）
  • 使用简单，你可以快速掌握。
  • 易于扩展 ，可自定义功能。

快速开始范例

只需要简单几行代码你就可以创建自己的交互式网络热力图：

	var heatmap = h337.create({
		container: domElement
	});

	heatmap.setData({
		max: 5,
		data: [{ x: 10, y: 15, value: 5}, ...]
	});
