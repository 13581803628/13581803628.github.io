此示例演示如何使用'addData'功能将数据动态添加到热力图实例上。
如果你点击灰色区域，将添加一个数据点。
你可以看到，如果热图的数据最大值改变，它可以自适应
（测试它：在同一位置点击两次）

演示

这里是非常有趣的点击区域！

代码


// 创建热力图实例
var heatmapInstance = h337.create({
    container: document.querySelector('.heatmap'),
    radius: 90
});
document.querySelector('.demo-wrapper').onclick = function(ev) {
    heatmapInstance.addData({
        x: ev.layerX,
        y: ev.layerY,
        value: 1
    });
};
// 就是这样... 准备好了吗？
